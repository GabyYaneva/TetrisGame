Tetris Classic Remake 🧩
⚠️ Project Status: Early Development
This project is currently in the alpha stage. I am focused on building the core engine, specifically the rotation logic (SRS) and the piece generation algorithm.

📝 Overview
A modern implementation of the iconic Tetris game. This project is a deep dive into grid-based collision detection, state management, and asynchronous game loops. The goal is to create a pixel-perfect recreation with smooth controls and classic mechanics.

🏗️ Technical Goals
Matrix Logic: Implementing a coordinate system to handle tetromino movement and rotation within a 10x20 grid.

Rotation System: Utilizing the Super Rotation System (SRS) to ensure fair and predictable piece behavior.

Performance: Ensuring minimal latency between user input and piece response.
